Installation Guide

1Download APK

Click the Android download button above to get the APK file.

2Enable Installation

Go to Settings → Security → Enable "Install from Unknown Sources" or allow when prompted.

3Install the App

Open the downloaded APK file from your Downloads folder and tap Install.

4Launch Mentora

Open the app from your home screen and start your career mentorship journey!

Note: This app is not from Google Play Store. Make sure you download from the official link only.
1Download IPA

Click the iOS download button above to get the IPA file.

2Install AltStore or Sideloadly

You'll need AltStore (free) or Sideloadly to install IPA files on iOS.

3Sideload the App

Connect your iPhone to computer, open AltStore/Sideloadly, and install the IPA file.

4Trust Developer

Go to Settings → General → Device Management → Trust the developer profile.

Note: Sideloaded apps need to be refreshed every 7 days with free Apple ID (or yearly with paid developer account).
